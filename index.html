Welcome to WebTopoTools, an exploratory tool for geomorphologist - currently in active development.

The tool is full client-side, meaning that everything is happening on your machine: no data transfer, gathering or costly server management. It is powered by webassembly, allowing us to deliver via the web a light version our research code base (in c++ and python) without any installation needed and nearly the same performances.

Few things to be aware of: this tool is at an experimental stage and might crash or softlock unexpectedly. If you refresh the page multiple times, reopen it in a new tab for a tru fresh start. There are technical limitations making the use of large files difficult (let's say >100Mb might become unstable) but stay tuned as this technology is evolving FAST. The software has been tested on Chrome and Firefox, MS Edge should be fine as based on chromium and Safari can be randomly unstable and buggy.

Send feedback/questions to boris.gailleton@gfz-potsdam.de
